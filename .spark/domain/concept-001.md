# deckboss-agent

## What It Does
- Optimize deck operations for fishing vessels
- Track crew assignments and deck logistics
- Monitor catch processing workflows
- Ensure safety protocols are followed
- Write tiles to PLATO with explicit schema

## PLATO Integration
This agent has an explicit PLATO tile schema (documented in agent docs).
Uses PLATO to coordinate with fleet about deck operations.
